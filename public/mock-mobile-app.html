🎯 仕様まとめ

技術選定: Capacitor (Ionic 系) で iOS / Android 同時開発。既存 Web 版 (app.html) を WebView でラップ + ネイティブ機能ブリッジ追加。

開発期間: MVP 2 週間 (Week 1: ラップ + ネイティブブリッジ / Week 2: プッシュ通知 + 配布)。ストア審査含めて 6-8 週間で公開。

必須実装: ログイン / チャット / 音声入力 / カメラ / プッシュ通知 (どれか1つ以上ないと App Store 審査落ち)

非対応: ブラウザ拡張 (モバイル Chrome 非対応) / マーケットプレイス出店フロー (PC 推奨、閲覧・購入は OK)
