🖥️ コンピュータ連携 (Local Agent)
エンドユーザー視点モック

ユーザーが自分の Mac/Windows PC を MY AI Agent に紐付けて、AI に シェル実行 / ファイル編集 / 画面操作 をさせるまでの全画面。
ペアリング → 権限付与 → 実利用 → 監査までを 9 ステップで描画。

💡 これは静的モックです。仕様書 #11 (基盤)・#12 (Code 互換)・#13 (Computer Use 互換) の挙動を可視化しています。

⚙ このデバイスに与える権限

AI が実行できる操作を選びます。あとから変更可。チェックを外せば AI はそのカテゴリの操作を一切できません。

⚡

毎回確認

~/projects/** のみ許可。.ssh / .env はブロック
自動許可
🌐

github.com, npmjs.org, registry-1.docker.io のみ許可
セッション中許可
📸
screen.capture — 画面キャプチャ
スクリーンショット取得 (Computer Use 用)
無効
🖱️
input.keyboard / input.mouse — 入力操作
キーボード打鍵・マウス操作 (高リスク)
無効

📋 ユーザーが触る画面のまとめ

上記9画面が、エンドユーザーが見る全部です。下記は一切やりません:

  • ❌ ターミナルで設定ファイル編集
  • ❌ コードビルド・コンパイル
  • ❌ 環境変数の設定
  • ❌ 鍵ペア生成 (Local Agent が自動で OS keychain に保存)
  • ❌ ファイアウォール設定 (Local Agent は outbound 接続のみ)

所要時間: 初回 5分 (ダウンロード + ペアリング + 権限選択) · 以降は常駐。クリック数: 約 8 回。

🔐 セキュリティ: Capability ベース権限 + 多層強制 + ハッシュチェーン監査ログ + 1クリック緊急停止。
Agent Storeで他人作の Agent を入れても、独立サンドボックスで動くので自分のホームディレクトリには触れません。
