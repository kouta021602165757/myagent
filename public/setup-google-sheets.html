📊 Google スプレッドシート連携
初期セットアップ

この画面はアプリ管理者（あなた）が1回だけやる作業です。終わったらエンドユーザーは「Google アカウントと接続」ボタンを押すだけで連携できます。
かかる時間: 5分。順番に下のステップを進めてください。

⚠️ なぜ手動なの?
Google が OAuth クライアント設定の API 変更を許可していないため、Google Cloud Console を1回だけ開く必要があります。スコープと redirect URI を追加すれば、以後永久に再設定不要です。
STEP 0

Google Sheets API を有効化

Google Cloud Project で Sheets API を ON にします。これをやらないと API 呼び出しで 403 エラーになります。1回だけの作業です。

🚀 Sheets API ライブラリを開く →
  1. 開いたページの右上で正しいプロジェクトが選ばれてるか確認 (OAuth クライアントを作ったプロジェクトと同じものを選ぶ)
  2. 青い 「有効にする」(ENABLE) ボタンをクリック
  3. ボタンが「管理」(MANAGE) に変わったら有効化完了
  4. 反映に 1〜2分 かかることがあります
STEP 1

OAuth クライアントに redirect URI を追加

Google Cloud Console の「OAuth クライアント」画面で、コールバック URL を追加します。

🚀 Google Cloud Console を開く →
  1. 左メニューの「認証情報」(または上のボタンで直接)を開く
  2. 既存の OAuth 2.0 クライアント ID をクリック
  3. 「承認済みのリダイレクト URI」セクションの「+ URI を追加」をクリック
  4. 下の URL をそのままコピペ:
  1. 画面下の「保存」をクリック
STEP 2

OAuth 同意画面にスコープを追加

「Google スプレッドシート読み書き」権限をユーザーに要求できるようにします。

🚀 OAuth 同意画面を開く →
  1. OAuth 同意画面 → 「編集」(または「アプリを編集」)
  2. 「スコープ」のステップまで進む(「次へ」「次へ」)
  3. 「+ スコープを追加または削除」をクリック
  4. 検索ボックスに以下を貼り付け:
  1. 表示されるスコープ(Google Sheets API · 表示と管理) にチェック → 「更新」
  2. 「次へ」「次へ」で保存

テストモード推奨 アプリの公開ステータスが「テスト中」なら、テストユーザー一覧に自分の Gmail を追加してください(同じ画面の「テストユーザー」セクション)。「本番」モードに上げるとスプレッドシートスコープは Google の verification 申請が必要です。個人利用なら test mode のまま使えます。

STEP 3

動作確認

ここまで終わったら、アプリに戻って実際に接続してみてください。

  1. アプリ画面で任意のエージェントを開いて「⚙ 編集」
  2. 「📊 Google スプレッドシート連携」セクションの「+ Google アカウントと接続」をクリック
  3. Google の同意画面で許可
  4. アプリに戻ってトーストに「✓ 接続中」が出れば完了
  5. 「📊」スイッチを ON にして保存
  6. そのエージェントとチャットして「このスプレッドシート(URL)を読み込んで」と頼んでみる

Q. これってエンドユーザーにもやってもらうの?

いいえ。STEP 1〜2 はアプリ管理者(あなた)が1回だけやる作業です。終わったらエンドユーザーは「Google アカウントと接続」ボタンを押すだけで、数秒で連携できます。

Q. 「redirect_uri_mismatch」エラーが出る

STEP 1 の URI が正しく登録されていません。コピーボタンで貼り直して再保存してください。よくある間違い: 末尾のスラッシュ、http と https の違い、サブドメイン違い(www. を付けるなど)。

Q. 「アクセスがブロックされました」エラーが出る

OAuth 同意画面が「テスト中」モードで、自分の Gmail がテストユーザーに登録されていません。STEP 2 の最後に書いたとおり、テストユーザー一覧に追加してください。

Q. 後から連携を解除したい

アプリのエージェント編集パネルから「切断」できます。Google 側でも一括で取り消せます: myaccount.google.com/permissions

Q. 「サーバー側の自動設定」って何?

Supabase の DB スキーマを自動マイグレーションする機能です。DATABASE_URL 環境変数を Render に追加していれば、起動時に自動で google_oauth 列が作られます。未設定の場合は手動で SQL を実行してください(Supabase Dashboard → SQL Editor → docs/SUPABASE_MIGRATION.sql の中身を実行)。
