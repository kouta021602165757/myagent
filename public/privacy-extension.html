MY AI Agent ブラウザ拡張
プライバシーポリシー

最終更新: 2026年5月7日 · 適用範囲: MY AI Agent — Browser Connector Chrome 拡張機能
サマリー (3 行)
① この拡張はあなたが MY AI Agent アプリで指示した時のみ動作します。
② 取得する情報は最小限: ペアリング用のデバイストークンと、各操作時の対象URL・操作種別のみ。
③ ページ内容は AI が処理する瞬間だけ取得し、終わったら破棄します。常時監視や履歴取得は行いません。

1. 取得する情報

本拡張機能 (以下「本拡張」) は以下の情報のみを取得・処理します:

項目取得タイミング用途
デバイストークン (32 バイトのランダム値)初回ペアリング時に1回サーバーがあなたのアカウントを識別するため
現在開いているタブの URL・タイトルAI ツール (ext_read_page) が呼ばれた瞬間のみAI に正しい操作を判断させるため
ページ内の表示テキスト (5,000 文字まで)同上同上
ページ内の操作可能要素 (ボタン・入力欄等の一覧)同上AI がクリック・入力対象を選定するため
スクリーンショット画像 (JPEG)AI が ext_screenshot を明示的に呼んだ時のみAI が視覚的判断を必要とする時のみ
各操作の実行結果 (成功/失敗・所要時間等)各操作実行時あなたへの結果報告・監査ログ

2. 取得しない情報

本拡張は以下の情報を取得しません:

  • あなたが入力したパスワード・クレジットカード番号・個人識別情報
  • 閲覧履歴 (AI が呼ばれていない時のページアクセス)
  • 常時バックグラウンドでのページ監視
  • Cookie の中身 (ブラウザの認証 Cookie はあなたの Chrome に残ったまま、本拡張が直接読み取ることはありません)
  • ブックマーク・ダウンロード履歴・他の拡張機能の情報

3. 情報の保存場所

情報保存場所保存期間
デバイストークンあなたの Chrome の chrome.storage.local (ローカルのみ)切断するまで
ページ操作の結果ログMY AI Agent サーバー (Render + Supabase, 日本国内リージョン)90日間 (監査用)
ページ内容・スクリーンショットサーバーで一時的に AI に渡した後即破棄非保存

4. 第三者への提供

本拡張が取得した情報は以下のサービスにのみ送信され、それ以外への第三者提供はありません:

  • MY AI Agent サーバー (myaiagents.agency) — 操作命令の中継と結果記録
  • Anthropic 社 (Claude API) — AI が応答を生成するため、ページの一部内容が API リクエストに含まれます

マーケティング目的での共有・販売は一切行いません。広告ネットワークへの提供もありません。

5. 暗号化と通信

  • サーバーとの全通信は HTTPS / TLS 1.3 で暗号化されます
  • デバイストークンは推測困難な 32 バイトのランダム値で、漏洩時はアプリ画面から即座に切断 (失効) 可能です
  • 本拡張のソースコードは GitHub で公開されており、誰でも検証可能です

6. ユーザーの権利

  • 切断: いつでも MY AI Agent アプリの「ブラウザ拡張連携」セクションから切断できます。トークンは即座に失効します。
  • アンインストール: chrome://extensions から本拡張を削除できます。削除と同時にローカル保存のトークンも消去されます。
  • ログ閲覧: アプリの「アクティビティ」タブで過去の全操作ログを閲覧できます。
  • 削除請求: 過去の操作ログ削除を希望される場合は kota.takeuchi@protocol.ooo までご連絡ください。30日以内に対応します。

7. 本拡張が必要とする権限の説明

権限必要な理由
tabsAI からの「URL を開く」「タブ一覧を取得」コマンドの実行
storageデバイストークンを chrome.storage.local に保存
scripting各ページに DOM 操作スクリプト (content.js) を注入
activeTabアクティブタブの内容取得・操作
notifications承認が必要な操作の OS 通知 (将来機能)
host_permissions: <all_urls>あらゆる Web サイト (X, Slack, Gmail, 自社 SaaS 等) で動作するため。ページ内容を読むのは AI ツールが呼ばれた瞬間のみ

8. 第三者サービスとの関係

本拡張がアクセスする可能性のあるサイト (X, Slack, Gmail 等) は、あなた自身がそれらのサービスにログインしている前提で動作します。本拡張がそれらのサイトにあなたとして接続することは、各サービスの利用規約上の「自動化ツール」「ボット」とみなされる可能性があります。各サイトの利用規約を遵守してご利用ください。

9. ポリシーの変更

本ポリシーは予告なく変更される場合があります。重要な変更がある場合はアプリ内通知でお知らせします。
