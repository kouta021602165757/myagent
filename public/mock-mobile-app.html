🎯 仕様まとめ

技術選定: Capacitor (Ionic 系) で iOS / Android 同時開発。既存 Web 版 (app.html) を WebView でラップ + ネイティブ機能ブリッジ追加。

開発期間: MVP 2 週間 / ストア審査含めて 6-8 週間で公開。

必須実装: ログイン / チャット / カメラ / プッシュ通知 / 下部タブバー (Apple のネイティブ機能要件クリア)

非対応: ブラウザ拡張連携 (モバイル Chrome 非対応) / Agent Store 出店フロー (PC 推奨、閲覧・購入は OK)
