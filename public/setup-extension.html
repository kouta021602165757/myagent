🌐 ブラウザ拡張をインストール

MY AI Agent ブラウザ拡張をインストールすると、AI がこの Chrome を直接操作できるようになります。
X 投稿、Slack 送信、Gmail 整理、社内 SaaS 自動化など ログイン済みサイトの操作はすべて可能。

📌 開発中 (BETA)
現在 Chrome Web Store 公開申請中です。それまでは 「デベロッパー モードで読み込み」 でインストールしてください。所要 1分。
STEP 1

拡張機能ソースをダウンロード

GitHub からソース一式 (ZIP) をダウンロードします。

⬇ GitHub から ZIP をダウンロード

解凍したフォルダの中の extension/ ディレクトリが拡張機能本体です。

STEP 2

Chrome に「読み込み済み」拡張として登録

Chrome のアドレスバーに以下を貼って Enter:

chrome://extensions
  1. 右上の 「デベロッパー モード」 トグルを ON
  2. 左上の 「パッケージ化されていない拡張機能を読み込む」 をクリック
  3. STEP 1 で解凍した extension/ フォルダを選択
  4. 拡張機能リストに「MY AI Agent — Browser Connector」が表示されたら成功
  5. 右上のパズルアイコン 🧩 →「MY AI Agent」のピン📌をクリック

ピン留めしておくと、ステータスがアイコンの色 (灰色=未連携 / 緑=接続中) で常に見えます。

STEP 3

拡張機能 ID をコピー

chrome://extensions 画面で MY AI Agent 拡張の 「ID」 をコピーしてください (32 文字の小文字英字)。

そして MY AI Agent アプリの「設定 → 詳細設定」に貼り付け 開発版のみ。
→ Chrome Web Store 公開後はこの手順は不要になります。

STEP 4

アプリ側でペアリング

  1. MY AI Agent アプリで任意のエージェントを編集
  2. 「🌐 ブラウザ拡張連携」セクションの「+ ブラウザ拡張をペアリング」をクリック
  3. 拡張アイコン 🧩 が緑色に変われば接続成功
  4. 同セクションのトグルを ON にして保存

Q. Chrome Web Store で公開はいつ?

申請中です。公開されればワンクリック (5秒) でインストール可能になり、上記 STEP 1〜3 が不要になります。それまでは開発者モードでお試しください。

Q. 何ができるの?

AI がこの Chrome を使って:

  • X (Twitter): 投稿・DM・いいね・フォロー
  • Slack: メッセージ送信・スレッド整理
  • Gmail: メール送信・下書き作成・整理
  • LinkedIn: 投稿・営業メッセージ
  • Notion / Salesforce / HubSpot: ログイン済み SaaS 全般
  • 社内ツール (独自Webアプリ): 既ログインなら何でも

Q. 安全?

はい、設計上以下を守っています:

  • サーバーから拡張への通信は SSE (TLS 1.3)
  • ペアリングトークンは 32 バイトのランダム値、ブラウザの localStorage 内
  • あなたのアカウントから発行されたトークンしか動作しない
  • 切断は chrome://extensions から拡張削除 OR アプリで「切断」ボタン
  • ホスト権限は <all_urls> ですが、AI が指示したときだけ動作

Q. 不可逆操作 (送信・削除) が怖い

承認モーダル (Phase 2) を順次実装中です。当面は AI に「投稿前は必ず確認」とプロンプト指示してありますが、最終的なボタンクリック前に必ず人間確認するワークフローを推奨します。

Q. アンインストールは?

chrome://extensions で「MY AI Agent」を「削除」するだけです。サーバー側のトークンも自動失効します。
