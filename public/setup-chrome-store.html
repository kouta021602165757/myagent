🚀 Chrome Web Store 公開ガイド

ブラウザ拡張を Chrome Web Store で公開する全手順。Takeuchi さん専用 (アプリ管理者向け)。
所要時間: 1時間 (申請まで) + 1〜3営業日 (Google 審査)。費用: $5 (一回限り)。

📌 やる順番が大事
① 開発者登録 → ② ZIP 準備 → ③ スクショ準備 → ④ ストアフォーム入力 → ⑤ 提出 → ⑥ 公開後の app.html 更新
各ステップに必要なもの・コピペ素材は全部このページにあります。
STEP 1

Chrome Web Store 開発者登録 ($5)

Google の Web Store 開発者として登録します (一回限り)。

🚀 Developer Dashboard を開く
  1. Google アカウントでログイン (Takeuchi さんの個人 or 会社 Gmail)
  2. 規約に同意
  3. $5 の登録料金支払い (クレカ)
  4. 登録完了 — Dashboard が見えるようになる
💡 Tip: 個人で出しても、会社 (PROTOCOL) で出してもOK。会社で出すなら Gmail Workspace アカウントから登録すると Publisher 名に「PROTOCOL Inc.」と表示できます。
STEP 2

拡張機能 ZIP を準備

ターミナルで以下を実行:

# プロジェクトルートで実行 cd ~/Desktop/myagent ./extension/build-zip.sh

→ プロジェクトルートに extension-v0.1.0.zip が生成されます。

💡 すでに作成済み: 私のほうで実行済みです。~/Desktop/myagent/extension-v0.1.0.zip にあります (12KB)。
Finder で開いて場所を確認してください。
STEP 3

スクリーンショット 5 枚を準備

サイズ: 1280 × 800 ピクセル。以下のページをブラウザで開いて、該当部分をキャプチャ:

  1. mock-extension-flow.html の STEP 02 (連携完了トースト)
  2. 同上 STEP 04 (X 投稿の承認モーダル)
  3. 同上 STEP 05 (進捗カード + 結果)
  4. mock-extension-sites.html の Slack ケース
  5. mock-extension-anysite.html の Uber ケース
💡 Mac でのキャプチャ: ⌘+Shift+4 → 範囲選択 → デスクトップに保存。Retina ディスプレイなら 2倍で保存されるので、プレビュー.app で「ツール → サイズを調整」で 1280×800 に縮小してから提出。
STEP 4

ストア掲載情報を入力

Developer Dashboard で「新しい拡張機能を追加」→ ZIP アップロード → 各フォームに以下をコピペ:

名前

概要 (132 文字以内)

プライバシーポリシー URL

サポートメール

公式サイト

詳細説明・各権限の説明文は長いので別ファイルにあります:

📋 docs/CHROME_STORE_LISTING.md を開く →

このファイルから:
・「詳細説明 (Description)」 (日本語版・英語版)
・「ホスト権限の説明」 6つ
・「単一目的の説明」
を順番にコピペしてください。

STEP 5

審査に提出

Dashboard 右上の「審査用に送信」ボタンをクリック → 完了。

提出前チェック
名前 / 概要 / 詳細説明 すべて入力済み
プライバシーポリシー URL を入力済み (提出前にアクセス可能か確認)
スクリーンショット 5 枚アップロード済み
単一目的の説明を入力済み
各権限の必要性を入力済み
「公開」モードを選択 (テスト中ではなく)
⏱ 審査期間: 通常 1〜3 営業日。Google から承認/却下のメールが届きます。
却下されたら: メールに具体的な理由が書いてあります。だいたい「権限の説明不足」「プライバシーポリシー不備」のどちらか。トラブルシュート集を参照して再提出してください。
STEP 6

公開後 - app.html に拡張機能 ID を埋め込む

承認されたら、Web Store の URL に拡張機能 ID が含まれます:

https://chrome.google.com/webstore/detail/[拡張機能 ID]?hl=ja

この 32 文字の小文字英字 をコピーして、以下のファイルに埋め込み:

  1. VS Code か Cursor で ~/Desktop/myagent/public/app.html を開く
  2. var EXT_ID = 'TBD'; を検索 (1箇所)
  3. var EXT_ID = 'あなたの拡張ID'; に書き換え
  4. 保存 → ターミナルで:
cd ~/Desktop/myagent git add public/app.html git commit -m "feat(ext): set Chrome Web Store extension ID" git push

→ Render が自動デプロイ → これで全エンドユーザーがワンクリックで使えるようになります。
