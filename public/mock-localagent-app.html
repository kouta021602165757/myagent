🏗️ 配布と仕組み

パッケージング: Tauri 2.0 で macOS .app / Windows .msi / Linux .AppImage を CI でクロスビルド。

サイズ: 単一バイナリ ~8MB (Electron の 1/15)。Rust ランタイム + システム WebView (macOS WebKit / Win WebView2)。

署名 + 公証: Apple Developer ID で署名 + notarize (Mac), Microsoft EV Authenticode (Windows)。両方とも CI で自動。

自動更新: 起動時に myaiagents.agency/api/agent/update をチェック → 新バージョンあれば差分配信 (Tauri Updater)。署名検証必須。

セキュリティ要件チェックリスト:

  • ✅ TLS 1.3 必須 + Server SPKI ピンニング
  • ✅ Ed25519 鍵ペアは macOS Keychain / Win Credential Manager / Linux libsecret に格納 (アプリ自身からも読めない)
  • ✅ Capability ベース権限 (UI層・Server層・Agent層 の 3層強制)
  • ✅ 60秒タイマー付き承認モーダル (クリックハイジャック耐性)
  • ✅ ハッシュチェーン監査ログ (改竄不可)
  • ✅ Outbound のみ接続 (NAT 越え不要 / firewall 設定不要)
  • ✅ 緊急停止: メニュー1クリック + ブラウザから1クリック + 全デバイス一括停止
  • ✅ サンドボックス: macOS App Sandbox + Linux bubblewrap + Windows AppContainer
  • ✅ 第三者製 Agent (Agent Store) は別プロファイルで完全隔離
